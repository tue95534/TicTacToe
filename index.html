About the game!

Simple Tic Tac Toe game. Levels easy and medium are very basic and easy to play. Hard level is challenging but winnable. There is a pattern you can use to always beat the game (see if you can figure it out!). The React version of this game (link end of page) does not allow wins in this same manner.
